UITableView

class UITableView : UIScrollView, NSCoding, UIDataSourceTranslating
  • register(cell:)

    Registers a WKTableViewCell object as cell if the cell view is into a xib. This method internally calls the register(_:forCellReuseIdentifier:) of the UITableVIew. Since it uses a UINib, the nibName must be the same of the class name!
